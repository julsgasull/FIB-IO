 (la tabla óptima)

variables artificiales i resolvedlo con el aplicativo, comprovamdo que efectivamente el 
problema auxiliar indica la infactibilidad del problema original

The exercise consists of solving a linear programming problem using the simplex algorithm

Select the LP problem that has been assigned to you from the collection of Lp problems in
LPs.zip

The delivery of this exercise consists of:

a) the solution as obtained usng the web page
http://www.zweigmedia.com/RealWorld/simplex.html
(the optimum tableau)

If no feasible solution exists, then create the auxiliary problem with artificial variables and solve it, checking 
that the auxiliary problem actually shows the infeasibility of the original problem.

b) reproduce by hand the last iteration of the algorithm as shown by the applet
   and check that the final tableau matches the one found in the solution, and
    at least, that the optimal value of the objective function equals the one reporte by LINDO.
   Take into account that the initial basic feasible solution is not shown by the applet. 
   If the solution is found in just one iteration, then the only tableau in canonical form shown will be the final one.
   In this case, form the auxiliary problem and solve it using the applet so that an initial basic feasible solution may 
   be used to start from.

c) check for alternative optima
d) In case that alternative optima exist, then calculate a basic one different from the that found by LINDO.
   Obtain also a non-basic solution for the problem using the previous one found by LINDO and the one that you have calculated.
